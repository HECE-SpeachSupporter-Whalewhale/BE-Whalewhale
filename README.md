## 📦주요 기능

### ⭐️ 위스와 발표하기 기능

- 기존에 있는 대본을 기반으로 사용자의 발표속도에 맞춰 대본을 넘겨줍니다

### ⭐️ 위스의 도움받기 기능

- 걱정마세요 대본작성에 적합하게 Fine-tuning된 AI(GPT-4o)에 기반하여 나에게 꼭 맞춘 대본을 만들어줍니다.

### ⭐️ 위스와 추억살펴보기 기능

- 대본이 100개가 넘어도 쉽게 내가 원하는 대본을 찾을 수 있는  즐겨찾기, 검색, 시간순 필터 설정으로 찾기 쉽게 정렬해 드립니다.
